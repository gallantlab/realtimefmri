Web interface
=============

There are many web endpoints to access and manipute parts of the real-time pipeline.

 * `Experiment`_
 * `Model`_

_`Experiment`
-------------
- ``/experiments``

  - List the available experiments


- ``/experiment/<name>/run``

  - Run the experiment


_`Model`
--------
 - ``model:<name>``


.. _redis: https://redis.io/documentation
